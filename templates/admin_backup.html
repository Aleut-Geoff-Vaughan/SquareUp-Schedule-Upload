Restore from backup

Replaces the current database with an uploaded backup file. The current database is saved to schedules.db.pre-restore.bak before being overwritten.

Warning: all current data (including your login) will be replaced with the contents of the uploaded file. You will need to log in again with credentials from the restored backup.
